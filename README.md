# ThereminWithArduino
Ultrasonik sensör kullanarak el mesafesini ses frekanslarına dönüştüren, Arduino tabanlı bir Theremin (Temassız Müzik Kutusu) projesi./ An Arduino-based Theremin (Contactless Music Box) project that translates hand distance into sound frequencies using an ultrasonic sensor.

## <a name="english"></a> English

### 📝 Project Description
This project is a digital version of the "Theremin," a musical instrument played without physical contact. It uses the **HC-SR04 Ultrasonic Distance Sensor** to detect hand movements and a **Buzzer** to generate corresponding tones. As the hand moves closer to the sensor, the sound frequency increases, creating a unique musical experience.

### 🛠 Hardware Components
- **Microcontroller:** Arduino Uno
- **Sensors:** HC-SR04 Ultrasonic Sensor
- **Output:** Passive/Active Buzzer
- **Connectors:** Jumper wires and Breadboard

### 🚀 Features
- Real-time distance-to-frequency mapping using the `map()` function.
- Operational range between 5cm and 30cm for optimal playability.
- Visual monitoring via Serial Monitor.

## <a name="turkish"></a> Türkçe

### 📝 Proje Hakkında
Bu proje, fiziksel temas olmadan çalınan "Theremin" müzik aletinin dijital bir versiyonudur. **HC-SR04 Ultrasonik Mesafe Sensörü** ile el hareketlerini algılar ve **Buzzer** aracılığıyla bu mesafeyi sese dönüştürür. El sensöre yaklaştıkça ses frekansı artar, böylece temassız bir müzik deneyimi sunar.

### 🛠 Kullanılan Malzemeler
- **Mikrodenetleyici:** Arduino Uno
- **Sensör:** HC-SR04 Ultrasonik Mesafe Sensörü
- **Çıkış:** Buzzer
- **Bağlantı:** Jumper kablolar ve Breadboard

### 🚀 Özellikler
- `map()` fonksiyonu kullanılarak gerçek zamanlı mesafeden frekansa dönüştürme.
- 5cm ile 30cm arasında optimize edilmiş çalma aralığı.
- Seri Port üzerinden anlık mesafe takibi.
